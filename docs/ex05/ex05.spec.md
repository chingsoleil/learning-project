## 一、技術規格

### 1.1 開發框架與工具
- **前端框架**: Vue 3 (Composition API)
- **CSS 框架**: Tailwind CSS
- **開發方式**: Mobile-First RWD（優先適配手機操作）
- **資料格式**: JSON（欄位命名參考 ex04/data-dictionary.md）
- **狀態管理**: Pinia（Vue 3 官方推薦）
- **圖表庫**: Chart.js / ECharts
- **路由**: Vue Router

### 1.2 響應式設計規格
- **Mobile（優先）**: 375x667 (iPhone SE), 390x844 (iPhone 13)
- **Tablet**: 768x1024 (iPad)
- **Desktop**: 1920x1080

### 1.3 色彩系統
- **主色 Primary**: #2C3E50 (深藍灰)
- **次色 Secondary**: #3498DB (天藍)
- **強調色 Accent**: #E74C3C (紅)
- **成功 Success**: #27AE60 (綠)
- **警告 Warning**: #F39C12 (橘)
- **錯誤 Error**: #E74C3C (紅)
- **背景 Background**: #ECF0F1 (淺灰)
- **卡片背景**: #FFFFFF (白)

### 1.4 互動規格
- 可點擊的按鈕與連結（明確的 hover/active 狀態）
- 表單輸入即時驗證與視覺回饋
- 頁面轉場效果（淡入淡出）
- 圖表動態呈現（進入視窗時動畫）
- Loading 狀態指示器
- Toast 通知訊息

## 二、前後端資料對應

### 2.1 欄位命名規範
- **前端 JSON**: 使用 camelCase（JavaScript 慣例）
- **後端資料庫**: 使用 PascalCase（ASP.NET Core 慣例）
- **API 層**: 自動進行命名轉換

### 2.2 主要資料表對應

| 前端命名 | 後端資料表 | 說明 |
|---------|-----------|------|
| testPaper | TestPaper | 測卷主檔 |
| questionBank | QuestionBank | 題庫主檔 |
| testSession | TestSession | 受測主檔 |
| testAnswer | TestAnswer | 作答明細 |
| instrumentCategory | InstrumentCategory | 量表分類 |
| traitCategory | TraitCategory | 特質面向分類 |

### 2.3 常用欄位對應

| 前端 (camelCase) | 後端 (PascalCase) | 資料表 | 說明 |
|-----------------|------------------|--------|------|
| id | Id | 所有表 | 主鍵 |
| testPaperId | TestPaperId | TestSession | 測卷ID |
| testTakerName | TestTakerName | TestSession | 受測者姓名 |
| phone | Phone | TestSession | 電話 |
| email | Email | TestSession | Email |
| birthDate | BirthDate | TestSession | 生日 |
| status | Status | TestSession | 狀態 |
| questionOrder | QuestionOrder | TestPaperDetail | 題目順序 |
| answerValue | AnswerValue | TestAnswer | 作答值 |
| textEn | TextEn | QuestionBank | 英文題目 |
| textZh | TextZh | QuestionBank | 中文題目 |

## 四、頁面架構及操作流程

### 4.1 使用者受測頁面

#### 4.1.1 首頁 (Home Page)
**路徑**: `/`

**內容元素**:
- 網站 Logo 與標題
- 導覽列: 首頁、測驗列表、查看報告、關於我們
- Hero Section:
  - 標題: "探索你的人格特質"
  - 副標題: "透過科學化的心理測驗，深入了解自己"
  - CTA 按鈕: "開始測驗"
- 測驗卡片列表:
  - 大五人格測驗
  - 職業興趣測驗（灰階，標示"即將推出"）
  - 情緒智商測驗（灰階，標示"即將推出"）
- 特色說明區塊:
  - 專業可靠
  - 即時報告
  - 隱私保護
- Footer: 聯絡資訊、隱私政策、服務條款

**互動行為**:
- 點擊 CTA 或測驗卡片 → 前往測驗介紹頁

---

#### 4.1.2 測驗介紹頁 (Test Introduction)
**路徑**: `/test/big-five-intro`

**內容元素**:
- 麵包屑導覽: 首頁 > 大五人格測驗
- 測驗標題: "大五人格測驗 (Big Five Personality Test)"
- 測驗介紹:
  - 簡介文字（200-300字）
  - 測驗理論背景
  - 五大人格維度說明卡片（含圖示）:
    * 經驗開放性 (Openness)
    * 嚴謹性 (Conscientiousness)
    * 外向性 (Extraversion)
    * 親和性 (Agreeableness)
    * 神經質 (Neuroticism)
- 測驗資訊:
  - 題目數量: 50 題
  - 預估時間: 10-15 分鐘
  - 適用對象: 18歲以上成人
- 注意事項:
  - 請在安靜環境下作答
  - 依直覺回答，無對錯之分
  - 所有資料將加密保護
- CTA 按鈕: "開始測驗"

**互動行為**:
- 點擊"開始測驗" → 前往個人資訊填寫頁

---

#### 4.1.3 個人資訊填寫頁 (User Information)
**路徑**: `/test/big-five/user-info`

**內容元素**:
- 進度指示器: 1.個人資訊 → 2.答題 → 3.結果
- 頁面標題: "請填寫基本資料"
- 表單欄位:
  - 姓名 (必填, text)
  - 性別 (必填, radio): 男性、女性、其他、不願透露
  - 年齡 (必填, number): 18-100
  - 電話號碼 (必填, tel): 格式驗證 09XX-XXX-XXX
  - Email (必填, email): 格式驗證
- 隱私聲明勾選框 (必填, checkbox): 
  "我已閱讀並同意隱私政策與使用條款"
- 按鈕:
  - "返回" (次要按鈕)
  - "下一步" (主要按鈕)

**驗證規則**:
- 所有必填欄位不可為空
- Email 格式驗證
- 電話號碼格式驗證
- 年齡範圍驗證

**互動行為**:
- 點擊"下一步" → 驗證通過後前往答題頁
- 驗證失敗 → 顯示錯誤訊息
- 點擊"返回" → 確認對話框 → 返回測驗介紹頁

---

#### 4.1.4 答題頁 (Test Questions)
**路徑**: `/test/big-five/questions`

**內容元素**:
- 進度指示器: 1.個人資訊 → **2.答題** → 3.結果
- 答題進度條: X / 50 題 (百分比視覺化)
- 當前題目顯示:
  - 題號: "第 1 題"
  - 題目文字（大字體，清晰易讀）
  - 答案選項（單選，5個選項）:
    * 非常不同意 (1分)
    * 不同意 (2分)
    * 中立 (3分)
    * 同意 (4分)
    * 非常同意 (5分)
- 導覽按鈕:
  - "上一題" (前往前一題，第1題時禁用)
  - "下一題" (前往下一題)
  - "送出測驗" (僅在第50題顯示)
- 輔助功能:
  - 題目清單側邊欄（可收合，Mobile 版為底部抽屜）
    * 顯示所有題號
    * 已答題標記為綠色
    * 未答題顯示為灰色
    * 點擊可直接跳轉
  - 答題狀態: "已完成 X / 50 題"

**答題規則**:
- 每題必須選擇一個答案才能繼續
- 可隨時返回修改答案
- 送出前未答完題目會顯示警告
- 答案自動儲存（LocalStorage 暫存）

**互動行為**:
- 選擇答案 → 高亮顯示，自動儲存至 LocalStorage
- 點擊"下一題" → 前往下一題
- 點擊"送出測驗" → 確認對話框 → 計算結果 → 前往結果頁
- 離開頁面 → 警告對話框（進度將不會儲存）
- Mobile: 支援左右滑動切換題目

---

#### 4.1.5 測驗結果頁 (Test Result)
**路徑**: `/test/big-five/result/:sessionId`

**內容元素**:
- 完成慶賀動畫 (checkmark animation)
- 結果標題: "測驗完成！"
- 結果概述:
  - 測驗名稱: 大五人格測驗
  - 完成時間: 2023-12-01 11:15
  - 測驗編號: Session #1（供日後查詢）
- 個人資訊摘要:
  - 姓名: 王小明
  - 性別: 男性
- 分數總覽（雷達圖）:
  - 五個維度的百分位數
  - 互動式圖表（hover/tap 顯示詳細數值）
  - Mobile: 響應式縮放
- 各維度分數卡片:
  ```
  [圖示] 經驗開放性
  分數: 38 / 50
  百分位: 75% (高於75%的人)
  評級: ★★★★☆ 高
  簡短描述: 您對新體驗持開放態度，富有想像力與創造力。
  ```
  (重複5個維度)
- 整體分析摘要（200-300字）
- 行動按鈕:
  - "查看完整報告" (主要按鈕)
  - "下載 PDF 報告" (次要按鈕，Mobile 版隱藏或移至選單)
  - "分享結果" (次要按鈕)
  - "重新測驗" (文字連結)

**互動行為**:
- 進入頁面時，圖表動態繪製
- Hover/Tap 圖表 → 顯示詳細數值
- 點擊"查看完整報告" → 前往詳細報告頁
- 點擊"下載 PDF" → 下載確認與載點
- 點擊"分享結果" → 顯示分享選項（複製連結、社群媒體）
