# cursor翻譯過程
```
docs\ex06\translation-process.md
```

- 要求使用claude sonnet4.5 直接翻譯，不要寫python程式翻譯。
- 原因：只有3000多行，沒有大到需要寫程式，但一次翻譯又多了點。
- 故要求分批直接翻譯。
- 翻譯品質並不完美，仍須人工校對。
- 一次處理過大量資料，可能導致結果有缺漏，或後半段出現嚴重幻覺。
- 所以應：分批執行，避免一次過多資料，並人工仔細校對。
