# 心得及疑問
- 如何確認測驗所需題目  (使用IPIP提供instrument或自行尋找其他測驗) 
- 
- CSV/SQL在系統中功能為何
  - CSV：原始題目資料，最終要匯入資料庫，成為可以操作的題目。
  - SQL：資料庫系統，負責提供資料服務
  - SQL server：提供服務的大樓，每個資料庫是一層樓，儲存不同資料，提供不同服務。
  - 每一個資料庫像一個excel檔，包含資料表(table)、欄位(column結構)、資料列(row內容)。
  - 今天產生的.sql檔分別執行：建立結構(create)、新增資料(insert)的任務。
  - CSV格式的題庫會被轉換成insert指令，以便新增到指定table裡。
