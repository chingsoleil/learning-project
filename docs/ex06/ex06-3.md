## 名詞解釋
### SQL
SQL（Structured Query Language，結構化查詢語言）是用於管理與操作關聯式資料庫的標準語言。
常見SQL指令類型如下：

### DDL
DDL（Data Definition Language）：定義管理資料庫結構（CREATE、ALTER、DROP）：建立與修改資料表、索引等。

### DML
DML（Data Manipulation Language）：操作資料
- 查詢資料（SELECT）：從資料表讀取資料
- 新增資料（INSERT）：將新記錄加入資料表
- 更新資料（UPDATE）：修改現有記錄
- 刪除資料（DELETE）：移除記錄
