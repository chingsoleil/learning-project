# 線上訂位（Reservations）企劃

- 路徑: `/reservations`
- 目標: 完成訂位提交，降低放棄率。
- 主要 CTA: 送出訂位。

內容大綱 / 表單欄位
- 姓名、電話、Email（選填）、用餐日期、時間、人數。
- 特殊需求（素食/過敏原/兒童座椅）、備註。
- 注意事項: 訂位保留時間、取消政策、訂金（如有）。

設計需求
- 表單分段與行動友善輸入（日期/時間/數字鍵盤）。
- 醒目但不刺眼的提交按鈕（深橘背景+白字）。

功能需求（後續可實作）
- 基本驗證、成功/失敗提示；選擇器（日期/時間）。

SEO
- 標題: 線上訂位｜品味軒
- 關鍵字: 訂位, 線上預約, 訂位政策

無障礙
- label 與 input 正確對應；錯誤訊息用文字提示不只顏色。

驗收清單
- [ ] 必填欄位清楚標示
- [ ] 成功提交後有回饋/導向
