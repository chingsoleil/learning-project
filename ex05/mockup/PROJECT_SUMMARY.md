## 📝 技術規格

### 前端技術棧
- **框架**: Vue 3.4+ (Composition API)
- **構建工具**: Vite 5.0+
- **路由**: Vue Router 4.2+
- **狀態管理**: Pinia 2.1+
- **樣式**: Tailwind CSS 3.4+ (Mobile-First)
- **圖表**: Chart.js 4.4+
- **字體**: Noto Sans TC (Google Fonts)

### 設計規範
- **最大寬度**: 1200px (Desktop)
- **斷點**: Mobile < 768px, Tablet 768-1919px, Desktop 1920px+
- **間距系統**: 8px grid
- **色彩系統**: Primary #2C3E50, Secondary #3498DB
- **最小觸控區域**: 44x44px (iOS), 48x48px (Android)

## 🧪 測試方式

### 使用者端測試流程
1. 啟動開發伺服器: `npm run dev`
2. 開啟瀏覽器: `http://localhost:3000`
3. 測試流程:
   - 首頁 → 測驗介紹 → 填寫資訊 → 答題（至少回答幾題測試進度）
   - 查看結果 → 詳細報告 → 填寫諮詢
   - 測試 Mobile/Desktop 響應式切換

### 管理系統測試
1. 前往登入頁: `http://localhost:3000/admin/login`
2. 使用測試帳號: `admin` / `admin123`
3. 測試各功能模組
4. 測試 Mobile 漢堡選單
