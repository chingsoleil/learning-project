# 精準修正翻譯的實用方法

## 方法：使用 Excel 篩選 + 人工審核

### 步驟 1：開啟檔案
在 Excel 中開啟 `IPIP_items-merged.csv`

### 步驟 2：建立檢查欄位（可選）
在最右側新增一欄 `review_status`，用於標記檢查狀態

### 步驟 3：使用篩選功能

**啟用自動篩選**：
- 選取標題列
- 資料 → 篩選

---

## 實用檢查方法

### 檢查 1：搜尋關鍵詞
點選 `text_zh` 欄位的篩選箭頭 → 文字篩選 → 包含

**建議搜尋的詞彙**：
```
1. 「。我」（檢查是否有太短的句子）
2. 「臉」（檢查是否應該是「面對」）
3. 「手」（檢查是否應該是「處理」）
4. 「冷」（檢查是否應該是「冷漠」）
5. 「熱」（檢查是否應該是「熱情」）
```

### 檢查 2：比對英文原文
在篩選結果中，逐一檢查 `text_en` 和 `text_zh` 是否語意相符

### 檢查 3：常見錯誤模式

| 英文 | 可能誤譯 | 正確翻譯 |
|------|---------|---------|
| "face my problems" | 我臉對我的問題 | 我面對我的問題 |
| "cold person" | 冷的人 | 冷漠的人 |
| "warm personality" | 溫暖的個性 | 熱情的個性 |
| "hand over" | 遞手 | 交出 |
| "feel down" | 感覺下 | 感到沮喪 |

## 快速操作技巧

### Excel 快捷鍵
- `Ctrl+F`：尋找
- `Ctrl+H`：取代
- `Ctrl+↓`：跳到下一個有內容的儲存格
- `Alt+↓`：開啟篩選下拉選單

### 建議工作流程

1. **第一輪**：篩選 `text_zh` 包含可疑詞彙
2. **第二輪**：檢查對應的 `text_en`
3. **第三輪**：修正翻譯
4. **第四輪**：清除篩選，檢查下一個詞彙

---

## 注意事項

⚠️ **不要使用「全部取代」**
- 必須逐一確認語境
- 同一個詞在不同句子中可能有不同意思

⚠️ **保持一致性**
- 相同英文應有相同中文翻譯
- 使用 Excel 的「排序」功能，將 `text_en` 相同的排在一起

⚠️ **備份檔案**
- 修改前複製一份備份
- 定期儲存
