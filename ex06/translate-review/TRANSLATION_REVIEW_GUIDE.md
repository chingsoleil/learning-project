## 常見問題詞彙清單

### 1. 字面翻譯（需檢查語境）

| 英文關鍵字 | 可能錯誤 | 正確意思（視語境） |
|-----------|---------|------------------|
| `blue/blues` | 藍色 | 憂鬱 |
| `face` | 臉 | 面對/面臨 |
| `hand` | 手 | 處理/應對 |
| `heart` | 心臟 | 內心/真心 |

| `down` | 下 | 沮喪/消沉 |
| `bright` | 明亮 | 聰明/樂觀 |
| `dark` | 黑暗 | 陰暗/消極 |

### 2. 檢查步驟

**在 Excel 中**：

1. **Ctrl+F** 開啟尋找
2. 在 `text_zh` 欄位搜尋可疑詞彙（如「藍色」、「臉」）
3. 逐一檢查英文原文，確認語境
4. 直接修正

**範例檢查指令**（在 `text_zh` 欄位搜尋）：
- 搜尋「藍色」→ 應該都是「憂鬱」
- 搜尋「臉」→ 可能是「面對」或「表情」
- 搜尋「手」→ 可能是「處理」

## 建議檢查順序

1. **搜尋顏色詞彙**：藍色、紅色、黑色、白色
2. **搜尋身體部位**：臉、手、頭、心臟
3. **搜尋溫度詞彙**：冷、熱、溫暖、寒冷
4. **檢查長度異常**：中文特別短或特別長的句子
5. **檢查標點符號**：是否使用全形標點

---

## 快速修正流程

### 使用 Excel 的「尋找與取代」

1. **Ctrl+H** 開啟取代視窗
2. **選項 → 範圍 → 選取欄位**（只在 `text_zh` 欄位替換）
3. 輸入要尋找的內容和取代內容
4. **逐一檢查**每個結果（按「尋找下一個」）
5. 確認無誤後按「取代」

**重要**：不要使用「全部取代」，必須逐一確認語境！
