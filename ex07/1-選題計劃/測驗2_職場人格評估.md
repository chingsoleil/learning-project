# 測驗 2：職場人格評估（Workplace Personality）

## 測驗基本資訊

| 欄位 | 值 | 說明 |
|------|-----|------|
| Name | 職場人格評估 | 測驗名稱 |
| Code | WORKPLACE_PERSONALITY_01 | 測驗代碼（唯一） |
| Description | 評估職場相關人格特質，聚焦職場表現相關特質，適合人才招募、團隊配置 | 測驗說明 |
| ResponseType | Likert5 | 作答方式：5點量表 |
| TimeLimit | 20 | 作答時間限制（分鐘） |
| TotalQuestions | 50 | 總題數 |
| IsActive | 1 | 是否啟用 |

## 使用的 Instruments

- **16PF** (InstrumentCategoryId: 1) - 16種人格因素問卷
- **HPI** (InstrumentCategoryId: 20) - Hogan人格量表
- **Goldberg1999** (InstrumentCategoryId: 17) - Goldberg大五人格量表

## 使用的 Traits

| Trait | TraitCategoryId | 目標題數 | 細分 Traits | 說明 |
|-------|----------------|---------|------------|------|
| Conscientiousness（盡責性） | 34 | 15題 | Achievement-striving (1), Dutifulness (52), Self-discipline (206) | 職場表現核心特質 |
| Extraversion（外向性） | 64 | 10題 | Assertiveness (16), Leadership (135) | 領導與溝通能力 |
| Agreeableness（親和性） | 8 | 10題 | Cooperation (38), Trust (235) | 團隊合作能力 |
| Emotional Stability（情緒穩定性） | 56 | 10題 | Anxiety (14) 反向計分 | 壓力承受能力 |
| Openness To Experience（開放性） | 162 | 5題 | Creativity/Originality (39) | 創新能力 |

**總計**：50題

## 選題條件

1. **InstrumentCategoryId** IN (1, 20, 17)
2. **TraitCategoryId** 優先順序：
   - Conscientiousness 相關：34, 1, 52, 206
   - Extraversion 相關：64, 16, 135
   - Agreeableness 相關：8, 38, 235
   - Emotional Stability 相關：56, 14（反向）
   - Openness 相關：162, 39
3. **Alpha >= 0.70**
4. **IsActive = 1**
5. **平衡性**：每個 trait 的正向題約 60-70%，反向題約 30-40%
