# 測驗 3：情緒與人際關係（Emotion & Interpersonal）

## 測驗基本資訊

| 欄位 | 值 | 說明 |
|------|-----|------|
| Name | 情緒與人際關係 | 測驗名稱 |
| Code | EMOTION_INTERPERSONAL_01 | 測驗代碼（唯一） |
| Description | 評估情緒管理與人際互動能力，深入評估情緒與人際面向，適合人際關係改善、情緒管理 | 測驗說明 |
| ResponseType | Likert5 | 作答方式：5點量表 |
| TimeLimit | 20 | 作答時間限制（分鐘） |
| TotalQuestions | 50 | 總題數 |

## 使用的 Instruments

- **HEXACO_PI** (InstrumentCategoryId: 18) - HEXACO人格量表
- **NEO** (InstrumentCategoryId: 26) - NEO人格量表
- **IPIP-IPC** (InstrumentCategoryId: 22) - IPIP人際特質量表

## 使用的 Traits

| Trait | TraitCategoryId | 目標題數 | 細分 Traits | 說明 |
|-------|----------------|---------|------------|------|
| Agreeableness（親和性） | 8 | 15題 | Empathy (58), Altruism (9), Cooperation (38), Trust (235) | 人際互動核心 |
| Neuroticism/Emotional Stability | 156/56 | 15題 | Anxiety (14), Depression (43), Emotional Stability (56) | 情緒管理 |
| Extraversion（外向性） | 64 | 10題 | Gregariousness (77), Friendliness | 社交能力 |
| Honesty/Integrity/Authenticity | 81 | 10題 | HEXACO 特有 | 誠實正直 |

**總計**：50題

## 選題條件

1. **InstrumentCategoryId** IN (18, 26, 22)
2. **TraitCategoryId** 優先順序：
   - Agreeableness 相關：8, 58, 9, 38, 235
   - Neuroticism/Emotional Stability：156, 14, 43, 56
   - Extraversion 相關：64, 77
   - Honesty/Integrity：81
3. **Alpha >= 0.70**
4. **IsActive = 1**
