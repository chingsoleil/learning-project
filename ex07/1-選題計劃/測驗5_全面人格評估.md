# 測驗 5：全面人格評估（Comprehensive Personality）

## 測驗基本資訊

| 欄位 | 值 | 說明 |
|------|-----|------|
| Name | 全面人格評估 | 測驗名稱 |
| Code | COMPREHENSIVE_PERSONALITY_01 | 測驗代碼（唯一） |
| Description | 多維度全面人格評估，最全面的人格評估，涵蓋多個理論模型，適合深度人格分析、研究用途 | 測驗說明 |
| ResponseType | Likert5 | 作答方式：5點量表 |
| TimeLimit | 20 | 作答時間限制（分鐘） |
| TotalQuestions | 50 | 總題數 |
| IsActive | 1 | 是否啟用 |

## 使用的 Instruments

- **HEXACO_PI** (InstrumentCategoryId: 18) - HEXACO人格量表
- **BFAS** (InstrumentCategoryId: 6) - 大五人格簡短量表
- **16PF** (InstrumentCategoryId: 1) - 16種人格因素問卷

| Creativity/Originality | 39 | 3題 | 細分特質 |

**總計**：50題

## 選題條件

1. **InstrumentCategoryId** IN (18, 6, 1)
2. **TraitCategoryId**：
   - 大五核心：64, 8, 34, 156, 162（各 6 題）
   - HEXACO 特有：81, 150（各 5 題）
   - 細分特質：135, 39 等（共 10 題）
3. **Alpha >= 0.70**
4. **IsActive = 1**
