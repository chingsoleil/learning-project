## 注意事項

⚠️ **題庫中所有題目的 ScoringKey 都是 1（正向題）**，沒有反向題（ScoringKey=-1）。這可能是題庫資料的特性，或者反向題的標記方式不同。在實際施測時，需要注意：

1. 所有題目都是正向計分
2. 某些題目的語意本身就是反向的（如「我不喜歡詩歌」），但計分方式仍為正向
3. 建議在計分時，根據題目語意判斷是否需要反向計分

```sql
-- 插入 TestPaper 並取得 Id
INSERT INTO TestPaper (Name, Code, Description, ResponseType, TimeLimit, TotalQuestions, IsActive)
VALUES (
    '大五人格快速評估',
    'BIG5_QUICK_01',
    '全面評估五大人格特質，適合一般人格評估、個人成長、職業規劃',
    'Likert5',
    20,
    50,
    1
);

DECLARE @TestPaperId INT;
SET @TestPaperId = SCOPE_IDENTITY();

-- 插入 TestPaperDetail
INSERT INTO TestPaperDetail (TestPaperId, QuestionBankId, QuestionOrder, IsRequired)
VALUES
    (@TestPaperId, 3120, 1, 1),
    (@TestPaperId, 3119, 2, 1),

    (@TestPaperId, 1297, 50, 1);
```
